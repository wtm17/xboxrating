About

Welcome to the Xbox 360 game voting application!

This application displays the games we currently own, the games we would like to buy, and the number of votes for each game. Each employee can vote for their favorite game or add a new game to this list once per day. At the end of the week, if we reach our productivity goals the game with the most votes will be purchased!

This application was written by Billy Meyers and primarily makes use of the following frameworks:

  • Groovy/Grails
  • AnglarJS
  • Twitter Bootstrap
